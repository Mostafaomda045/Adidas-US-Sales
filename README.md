# Adidas US Interactive Sales Dashboard

This repository contains an interactive sales dashboard visualizing Adidas US sales data across regions, methods, gender demographics, product categories, and quarterly trends. The dashboard provides key insights into profitability, sales distribution, and growth patterns to support data-driven business decisions.

## Dashboard Overview

The dashboard features a multi-panel layout displaying interconnected KPIs and trends for Adidas US operations:
- **Profit by Region**: Bar chart highlighting total profits across key states (Alaska, California, New York, Texas).
- **Sales Method Distribution**: Pie chart showing proportions of retail vs. online sales.
- **Gender Sales Breakdown**: Pie chart illustrating male vs. female product sales shares.
- **Sales Trend by Quarter**: Line graph tracking sales progression over time (e.g., Q1 to Q4).
- **Product Category Sales**: Additional pie chart for product-specific distributions.

Data is sourced from US sales records, enabling interactive filtering by region and time period for deeper analysis.

## Technologies Used
- **Visualization**: Power BI (inferred from interactive layout and styling).
- **Data Source**: Adidas US transactional database (regions, sales methods, demographics, quarters).
- **Interactivity**: Filters for states (sidebar) and dynamic updates across panels.

## Usage Instructions

1. Clone the repository.
2. Open `Adidas_US_Sales_Dashboard.pbix` in Power BI Desktop.
3. Refresh data connections for latest figures.
4. Interact via region slicers or date filters for custom views.

For contributions or customizations, submit pull requests with updated visuals or datasets.
